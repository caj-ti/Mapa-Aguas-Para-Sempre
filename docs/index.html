Bem-vindo(a) ao Mapa do Programa Águas Para Sempre

Este mapa exibe os limites territoriais e as propriedades associadas ao Programa Águas para Sempre na região rural de Joinville e Garuva, para análise e apresentação. As propriedades são classificadas em: Aderidas, Em processo de adesão e Interessadas / Manifestaram interesse.

O botão Painel mostra os dados agregados do grupo selecionado. A seleção considera apenas propriedades ativas no mapa. Propriedades desmarcadas na legenda não entram nos cálculos. Propriedades selecionadas são destacadas em vermelho. Passar o cursor sobre uma feição altera apenas a visualização e não os cálculos.

O botão Gráficos exibe os gráficos disponíveis.

O mapa é atualizado conforme necessário para refletir o estado atual do programa.
